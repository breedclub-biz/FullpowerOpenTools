# 色の使い分け

| サイト | アクセント色 | 配色の主役 |
|---|---|---|
| 買取LP（kaitori/） | **#cc0000（赤）** | WANTEDの強い赤、ヘッダー赤、モーダル赤 |
| ルートページ（/index.html） | **#FF5797（FP Pink）** | デザインシステム準拠 |
| 規約LP（fullpower repo: docs/design/） | **#FF5797（FP Pink）** | デザインシステム準拠 |

## なぜ買取LPだけ赤を維持しているのか

**Why**：
- 買取LPの「WANTED」赤は強い意匠性を持ち、視認性とブランド印象に寄与している
- 公式マスコット「ランテ」彩色準拠でアクセントを #FF5797 に統一する方針が決まったが、買取LPの赤を一律でピンクに置き換えると「WANTED」表現の迫力が薄れる
- 2026-05-04 のデザインシステム公開時に、ユーザー判断で「買取LPの WANTED 等の赤は維持」を確定

**How to apply**：
- 買取LP内の赤関連スタイル（#cc0000・#800000・#e74c3c 等）は触らない
- 買取LP以外で新規ページ・コンポーネントを作る場合は #FF5797 をアクセントにする
- 将来「色を統一すべきでは？」という議論が出たときは、この記録を参照しユーザーに再確認する
